# 社内FAQ

## 勤怠・休暇

### 有給休暇
- 年間20日付与されます
- 入社6ヶ月後から取得可能です
- 申請はSlackの #申請 チャンネルで行います
- 半日単位での取得も可能です

### 残業
- 月45時間を超える残業は事前申請が必要です
- 深夜残業（22時以降）は別途手当が支給されます

### リモートワーク
- 週3日まで可能です
- 申請は前日までにSlackで行います
- コアタイムは10:00〜15:00です

## 経費・給与

### 経費精算
- MoneyForwardで申請します
- 領収書は30日以内に提出してください
- 上限は1件あたり10万円です

### 交通費
- 通勤交通費は月額上限3万円まで支給されます
- 出張交通費は全額支給されます

### 給与
- 毎月25日払いです
- 締め日は月末です

## 設備・環境

### PC
- 入社時にMacBook Proが支給されます
- 故障時はIT部門に連絡してください

### 健康診断
- 毎年6月に実施されます
- 受診は全社員必須です
- 費用は会社負担です
